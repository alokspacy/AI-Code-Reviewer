# AI Code Reviewer

AI Code Reviewer is an AI-powered GitHub integration that automatically analyzes pull requests and generates actionable review comments to help developers write cleaner, more maintainable code.

It helps identify potential bugs, code smells, security concerns, and improvement opportunities before code is merged.

## Features

- Automated pull request reviews
- AI-generated code quality suggestions
- Bug and issue detection
- Security-focused recommendations
- Inline review comments on pull requests
- GitHub Actions integration
- Configurable review behavior
- Multi-language support

```text
Developer → Pull Request
                  ↓
          AI Code Reviewer
                  ↓
